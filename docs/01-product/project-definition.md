# VinkoClaw Project Definition v0.1

## Positioning

VinkoClaw is a local-first OPC AI team operating system built for DGX Spark.

It is not a general chatbot and not a generic agent platform. It is an owner-facing operating layer for running a one-person company with an AI team behind it.

## Core Idea

The Feishu group is the command room.

The DGX Spark machine is the local AI headquarters.

The owner gives instructions, checks progress, approves risky actions, and receives results. Multiple internal AI roles cooperate behind a single Feishu bot interface.

## Target Users

- One-person company founders
- Developer-founders
- Researchers and knowledge workers
- Operators who want a local AI team instead of a cloud-only assistant

## Product Value

- Remote control of a local AI team
- Local execution on DGX Spark
- Multi-role collaboration instead of single-assistant chat
- Clear task ownership, state, and approvals
- Local knowledge over code, docs, PDFs, and notes

## First-Phase Scope

The first version should prove one thing:

One owner can manage a working AI team through Feishu, while the actual work happens on the local DGX Spark machine.

## MVP Outcomes

- Feishu group interaction works
- The system exposes multiple AI roles
- Code and document knowledge can be used locally
- Tasks can be assigned, tracked, and reported
- Drafted external communication can be reviewed before approval
